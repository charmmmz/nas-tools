Default Behavior

Use Locale.autoupdatingCurrent.region?.identifier so the app follows the user's current iOS Language & Region setting. No location permission is needed.

User Override

The region button opens a sheet. Automatic stores no region override; choosing US, JP, etc. stores that explicit TMDB region for future home loads.

Backend Contract

iOS sends a two-letter ISO region code to NASTOOL. Backend validates it and passes it to TMDB as watch_region for Streaming and region for Theaters.
