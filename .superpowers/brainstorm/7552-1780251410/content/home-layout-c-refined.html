Trending

Segmented by time window, using TMDB trending through NASTOOL.

  • Today: TMDB /trending/all/day
  • This Week: TMDB /trending/all/week

Popular

Segmented by availability/context, using Discover or existing movie APIs through NASTOOL.

  • Streaming: Discover movie + TV with watch_region and with_watch_monetization_types=flatrate
  • Theaters: Discover movie with region and with_release_type=2|3; movie-only by definition

Backend Shape

iOS calls one NASTOOL endpoint, not TMDB directly. The response keeps the current card fields plus optional section, source, and paging metadata.
